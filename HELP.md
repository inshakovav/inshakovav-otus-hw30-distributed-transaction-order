#### Get all Orders
```bash
curl http://localhost:8000/order | json_pp
```

#### Add an Order
```bash
curl -H 'Content-Type: application/json' \
     -d '{ "description":"Order description"}' \
     -X POST \
     http://localhost:8000/order | json_pp
```
